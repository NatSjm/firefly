# Proposal: add-lost-fireflies

Introduce the Lost Fireflies slice as a tracked retrofitted OpenSpec change for the existing lost-request capability.

This change captures the already-implemented Lost Fireflies requests feature (public browse, city/type filters, authenticated create, detail page with mailto contact) so the slice can move through Phase 4 with explicit tests, review evidence, and archive history.

**FRs covered:** FR-LOST-01, FR-LOST-02, FR-LOST-03, FR-LOST-04, FR-LOST-05.
