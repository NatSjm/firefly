# Proposal: add-identity-and-access

Introduce the identity and access slice as a tracked retrofitted OpenSpec change for the existing auth and shell capability.

This change captures the already-implemented registration, login, JWT-authenticated profile flow, protected routing, and authenticated shell behavior so the slice can move through Phase 4 with explicit tests, review evidence, and archive history.

**FRs covered:** FR-AUTH-01, FR-AUTH-02, FR-AUTH-03, FR-AUTH-04, FR-AUTH-05, FR-SHELL-01, FR-SHELL-02, FR-SHELL-03, FR-SHELL-04.
