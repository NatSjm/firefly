# Proposal: add-personal-archive

Introduce the personal archive slice as a tracked retrofitted OpenSpec change for the existing memory CRUD capability.

This change captures the already-implemented memory creation (story and recipe), photo upload, privacy control, dashboard filtering, editing, deletion, and single-memory viewing so the slice can move through Phase 4 with explicit tests, review evidence, and archive history.

**FRs covered:** FR-MEM-01, FR-MEM-02, FR-MEM-03, FR-MEM-04, FR-MEM-05, FR-MEM-06, FR-TOPIC-01, FR-TOPIC-02, FR-CITY-01.
