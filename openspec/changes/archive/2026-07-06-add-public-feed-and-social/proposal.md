# Proposal: add-public-feed-and-social

Introduce the public feed and social slice as a tracked retrofitted OpenSpec change for the existing feed, likes, and comments capability.

This change captures the already-implemented public memory feed (city/topic/sort filters, pagination), warmth/like toggle, and comment CRUD so the slice can move through Phase 4 with explicit tests, review evidence, and archive history.

**FRs covered:** FR-FEED-01, FR-FEED-02, FR-FEED-03, FR-FEED-04, FR-FEED-05, FR-FEED-06, FR-FEED-07, FR-LIKE-01, FR-LIKE-02, FR-COMMENT-01, FR-COMMENT-02, FR-CITY-02.
